# AniWifeProtect-Bot

### Telegram Channel: @AniWifeHub
### Telegram Bot: @AniWifeProtectBot

## command:

### default
- /start - send start message

### group:
 - .mute - mute a user by reply on user message
 - .unmute - unmute a user by reply on user message
 - .list.mute - list muted users from group
 - .ban - ban a user by reply on user message
 - .unban - unban a user by reply on user message
 - .list.ban - list banned users from group
 - .pin - pin a message by reply on a message
 - .unpin - unpin a message by reply on a message

### staff:
- .staff.add.admin - add admin by reply on user message
- .staff.remove.admin - remove admin by reply on user message
- .staff.add.moderator - add moderator by reply on user message
- .staff.remove.moderator - remove moderator by reply on user message

## setup
 - <code>git clone https://github.com/AmirrezaJalilian/AniWifeProtect-Bot.git </code>
 - <code>cd AniWifeProtect-Bot</code>
 - <code>pip install python-telegram-bot</code>
 - <code>python bot.py</code>
